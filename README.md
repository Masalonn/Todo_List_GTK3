LISTA ZADAŃ (C++ & GTK3)

Aplikacja desktopowa do zarządzania zadaniami z zapisem do pliku.

TECHNOLOGIE
C++17, GTK3, RapidJSON oraz Glade XML.

FUNKCJE
Zapis i odczyt listy z pliku data.json przy użyciu biblioteki RapidJSON. Wyszukiwanie zadań w czasie rzeczywistym oraz sortowanie według priorytetów lub nazwy. Kolorowanie zadań przez style CSS w zależności od ich ważności. Separacja interfejsu w pliku .glade od logiki napisanej w C++.
