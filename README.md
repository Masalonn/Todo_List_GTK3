TASK MANAGER (C++ & GTK3)
A desktop task management application featuring persistent storage and a dynamic user interface.

PROJECT OVERVIEW
A lightweight C++ application designed for efficient task organization, utilizing JSON for data persistence and a decoupled UI architecture.

TECHNOLOGIES
Language: C++17

UI Framework: GTK3 (with Glade XML for UI definition)

Data Format: JSON (via RapidJSON library)

Styling: Custom CSS for GTK

KEY FEATURES
JSON Persistence: Robust reading and writing of task lists to data.json using RapidJSON library.

Real-time Interaction: Integrated live search functionality and dynamic sorting by priority or task name.

Visual Prioritization: Implementation of CSS-based styling to color-code tasks according to their importance levels.

Separation of Concerns: Clean architectural split between the UI definition (.glade files) and the core application logic in C++.
